# Md. Ammar Hossain's Portfolio
This document describes the structure, features, and technologies used in the personal portfolio website of me. A modern, fully responsive personal portfolio website built using HTML, Tailwind CSS, JavaScript, and Cloud-hosted assets. This website showcases my background, projects, skills, achievements, and contact information in a visually appealing and user-friendly manner. 

## Features
- Modern UI/UX with Tailwind CSS
- Dark/Light mode support
- Smooth scrolling
- Form validation with popup messages
- Project showcase
- Skills section with progress bars
- Contact form using FormSubmit
- Animated section transitions
- Fully responsive for mobile, tablet, and desktop

## Sections
- Home
- About Me
- Projects
- Skills
- Contact
